#### For Development
######A. Run frontend dev server
1. npm install
2. npm run build
3. npm start

######B. Run backend server
1. mvn clean
2. mvn spring-boot:run

See live load in localhost:3000

#### For Production
1. npm install
2. npm run build
3. mvn clean
4. mvn package
5. java -jar app-0.0.1.jar
